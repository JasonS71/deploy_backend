# proyecto backend con nestJS

## Comandos de Instalacion y Ejecucion
```
# Navegar al directorio
cd backend2

# Instalar dependencias
npm install --save @nestjs/passport passport passport-local passport-jwt @nestjs/jwt
npm install --save-dev @types/passport-local @types/passport-jwt
npm install --save class-validator class-transformer
npm install --save @nestjs/swagger swagger-ui-express
npm install --save bcrypt
npm install --save-dev @types/bcrypt
npm install --save prisma @prisma/client
npm install --save-dev @nestjs/config

# Para ejecutar el servidor en modo desarrollo
npm run start:dev

# Para construir el proyecto
npm run build

# Para ejecutar el proyecto en producci√≥n
npm run start:prod
```
